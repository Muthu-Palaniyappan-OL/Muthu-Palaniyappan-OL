# Preact Template with Tailwind CSS and Webpack

This is a template project for building web applications with [Preact](https://preactjs.com/), [Tailwind CSS](https://tailwindcss.com/), and [Webpack](https://webpack.js.org/).

## Features

- **Preact**: A fast and lightweight alternative to React for building user interfaces.
- **Tailwind CSS**: A highly customizable CSS framework for rapidly building modern designs.
- **Webpack**: A popular module bundler for optimizing and bundling JavaScript, CSS, and other assets.

## Getting Started

To use this template, follow these steps:

1. Clone this repository:

   ```shell
   git clone https://github.com/Muthu-Palaniyappan-OL/preact-tailwind-template.git
   ```

2. Navigate to the project directory:

   ```shell
    cd preact-tailwind-template
   ```

3. Install the dependencies:

   ```shell
   npm install
   ```

4. Start the development server:

   ```shell
   npm run start
   ```

   This will start the development server and open the application in your default browser. Any changes you make to the source code will be automatically reloaded.

5. Build the production-ready assets:

   ```shell
   npm run build
   ```

   This command will generate optimized assets in the dist directory. These assets are ready to be deployed to a production environment.
